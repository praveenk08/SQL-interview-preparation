### Day 1: SELECT Statement

Today, I learned the foundation of SQL — the `SELECT` statement.

The SELECT statement is used to retrieve data from a database table and is one of the most frequently used SQL commands.

Example:

```sql
SELECT employee_name, salary
FROM Employees;
```

### Key Learnings

✅ Retrieve data from tables

✅ Select specific columns

✅ Use aliases for better readability

✅ Understand why `SELECT *` should be avoided in production
